## 1. General Information

&emsp;**Event Name:** FCAJ Community Day - June 2026

&emsp;**Date:** 27/06/2026

&emsp;**Location:** Online

&emsp;**Role:** Attendee

&emsp;**Format:** Online webinar featuring technical sessions on Cloud, AI, and DevOps advancements.

## 2. Main Content of the Event

The event featured a series of technical presentations and discussions focused on Cloud, AI, and DevOps advancements:

1. **Cloud Operations & Startups** - Speaker: Steve Trần from Cloud Thinker
   
   This session discussed the evolution of FinOps and CloudOps, emphasizing the importance of transitioning from human-heavy manual tasks to AI-driven automation, especially for large enterprises.

2. **Voice AI Agents** - Speakers: Nghị and Kiệt from Renova Cloud and Student Video Group
   
   This talk explored the architecture of Voice AI. The speakers highlighted the challenges of using speech-to-speech models for low-resource languages like Vietnamese and proposed using Tool Calling with LLMs as a more robust and reliable solution for real-world business applications.

3. **DevOps Agent** - Speakers: Nguyên and Bảo from Cloud Kinetis
   
   The speakers demonstrated a DevOps Agent that helps engineers investigate and resolve cloud infrastructure issues. They showcased how the agent can analyze logs, identify root causes, and suggest mitigation plans to significantly reduce MTTR (Mean Time To Repair).

4. **AI in HR** - Speakers: Trường and Minh Anh from Noventis
   
   This presentation discussed the integration of AI in Human Resources, specifically demonstrating how Amazon Q can assist HR professionals by automating candidate screening, CV reviewing, and JD creation, allowing them to focus on strategic decision-making.

5. **Amazon Q Security** - Speakers: Toàn Nguyễn and Nghị
   
   The final session covered the implementation of private security for Amazon Q in enterprise environments, ensuring secure integration with third-party systems.

## 3. Key Takeaways & Personal Contribution

Attending this online event provided me with valuable insights and lessons:
- **AI-driven Automation in Operations:** The transition from manual processes to AI-driven automation in CloudOps and FinOps highlights how modern enterprises optimize resource usage and reduce operational overhead.
- **Pragmatic AI Architectures:** The discussion on Voice AI agents showed that using Tool Calling with LLMs is often more reliable than complex speech-to-speech models for low-resource languages like Vietnamese, offering a more stable path for business applications.
- **Improving Operations Efficiency:** The DevOps Agent demonstration showed the practical application of LLMs in root-cause analysis and log processing, proving that automated mitigation suggestions can significantly reduce MTTR and improve system reliability.
- **Enterprise-grade AI Security:** Learning about private security implementation for Amazon Q emphasized the importance of secure data ingestion and API integration when deploying AI tools in highly regulated enterprise environments.

Although I participated online, I actively engaged with the chat and followed the Q&A sessions, particularly asking about best practices for structuring Tool Calling schemas and ensuring data privacy when connecting enterprise agents to third-party databases.
