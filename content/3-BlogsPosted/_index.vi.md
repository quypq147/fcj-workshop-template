Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

### [Blog 1 - ĐƠN GIẢN HÓA TÍCH HỢP AWS APPSYNC EVENTS VỚI POWERTOOLS FOR AWS LAMBDA](3.1-Blog1/)
Blog này giới thiệu Powertools for AWS Lambda vừa ra mắt tính năng mới AppSyncEventsResolver (hỗ trợ Python, TypeScript và .NET), giúp đơn giản hóa việc tích hợp AWS AppSync Events cho các ứng dụng thời gian thực như ứng dụng chat, bảng điều khiển hoặc hệ thống IoT.

### [Blog 2 - Đột phá stale knowledge của AI Agent với Web Search trên Amazon Bedrock AgentCore](3.2-Blog2/)
Bài viết giới thiệu về tính năng Web Search vừa được General Availability (GA) trên Amazon Bedrock AgentCore, giải pháp fully managed và tương thích MCP giúp các AI Agent khắc phục giới hạn kiến trúc tĩnh bằng cách tìm kiếm thông tin theo thời gian thực (real-time).

### [Blog 3 - Hướng dẫn tối ưu hóa Quản lý Đa ứng dụng (Multi-App) với Unified Webhooks trên AWS Amplify Hosting](3.3-Blog3/)
Bài viết chia sẻ cách tính năng Unified Webhooks trên AWS Amplify Hosting giải quyết bài toán giới hạn số lượng webhook từ các Git Provider (GitHub, GitLab, Bitbucket), đặc biệt hữu ích khi quản lý và triển khai các dự án có kiến trúc monorepo phức tạp.
