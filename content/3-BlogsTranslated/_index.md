### [Blog 1 - Simplify AWS AppSync Events integration with Powertools for AWS Lambda](3.1-Blog1/)
This blog introduces a new AppSyncEventsResolver feature in Powertools for AWS Lambda (supported in Python, TypeScript, and .NET), which simplifies integration with AWS AppSync Events for real-time applications such as chat apps, dashboards, or IoT systems.

### [Blog 2 - Breaking AI Agent Stale Knowledge with Web Search on Amazon Bedrock AgentCore](3.2-Blog2/)
This post introduces the newly General Availability (GA) Web Search capability on Amazon Bedrock AgentCore, a fully managed, MCP-compatible solution that enables AI Agents to bypass frozen training data by accessing real-time web search results.

### [Blog 3 - Simplifying Multi-App Management in AWS Amplify Hosting with Unified Webhooks](3.3-Blog3/)
This post covers how Unified Webhooks on AWS Amplify Hosting resolves git provider webhook limits for monorepos, allowing team developers to connect multiple Amplify applications to a single repository with a single unified webhook.
